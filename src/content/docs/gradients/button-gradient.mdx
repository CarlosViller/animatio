### Steps

1. Create your background gradient; you can use cssgradient.io for that.
2. Apply the background to an element (e.g., a button) and set the background width to 200%. We do this to use only half of our gradient, and when we hover over our element, we will use the other half:

```css
#button-example {
  margin: 20px auto;
  display: block;
  padding: 10px 20px;
  font-size: 1.5em;
  border-radius: 10px;
  border: none;
  /* ðŸ‘‡ */
  background-image: linear-gradient(
    90deg,
    rgba(131, 58, 180, 1) 0%,
    rgba(253, 29, 29, 1) 50%,
    rgba(252, 176, 69, 1) 100%
  );
  background-size: 200% auto;
  transition: background-position ease 500ms;
}
```
3. Move the background position on hover to the right:

```css
#button-example:hover {
  background-position: right center;
}
```

Try this effect yourself; you can do it from left to right, from right to left, or from top to bottom. Experiment!
